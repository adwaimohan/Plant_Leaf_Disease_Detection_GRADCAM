# 🌿 Crop Disease Detection using EfficientNet + Grad-CAM

## 🚀 Features
- Image-based plant disease detection
- EfficientNet-B0 model
- Grad-CAM visualization for explainability
- Streamlit web app

## 🧠 Tech Stack
- PyTorch
- Streamlit
- OpenCV
- NumPy

## 📦 Setup

```bash
pip install -r requirements.txt
